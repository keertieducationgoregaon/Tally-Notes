1. Introduction to Inventory in Tally Prime

Inventory refers to all goods held for sale, consumption, or production.

Tally Prime provides a full inventory management system for:

  • Stock Tracking
  • Godown Management
  • Purchase & Sales Management
  • Manufacturing Processes
  • Item-wise Profit Analysis

Inventory is built using three master components:

  1. Stock Groups
  2. Units of Measure
  3. Stock Items

These must be created before entering any inventory voucher.

2. Understanding Inventory Masters

MasterPurpose
Stock GroupsClassification of items (e.g., Mobiles, Accessories)
Units of MeasureDefine how quantity is measured (Nos, Kg, Box)
Stock ItemsActual products (Samsung A15, JBL Earphones)

3. Stock Groups (Theory + Practical)

Stock Group is a classification of products based on similar characteristics.

Benefits:

  • Category-wise reporting
  • Sales/profit analysis
  • Bulk pricing & tracking

How to Create Stock Groups

Method 1: Single Creation

  1. Gateway → Chart of Accounts → Inventory Masters → Stock Groups → Create
  2. Name: e.g., Electronics
  3. Under: Primary (or another group)
  4. Save (Ctrl + A)

Method 2: Multiple Creation

Chart of Accounts → Stock Groups → Create Multiple → Select parent → Enter list → Save

4. Units of Measure (Theory + Practical)

Units define how quantity is measured.

TypeExample
Simple UnitNos, Kg, Ltr
Compound Unit1 Box = 12 Pcs

How to Create Units

  1. Gateway → Chart of Accounts → Units → Create
  2. Type: Simple / Compound
  3. Symbol: Nos
  4. Formal Name: Numbers
  5. Decimal Places: 0
  6. Save

5. Stock Items (Theory + Practical)

Stock Item = Actual product in inventory.

Required for all inventory transactions.

How to Create Stock Items

  1. Gateway → Chart of Accounts → Stock Items → Create
  2. Name: e.g., Samsung A15
  3. Under: Mobiles (Stock Group)
  4. Unit: Nos
  5. Opening Quantity (optional)
  6. Save

Multiple Creation also available via Create Multiple option.

6. Inventory Creation Rules

  1. Every Stock Item must belong to a Stock Group
  2. Units are mandatory
  3. Use simple units unless required
  4. Use clean, short product names
  5. No duplicate item names
  6. Opening stock must be accurate

7. Business-Wise Inventory Examples

Trading Business (Electronics)

  • Groups: Electronics → Mobiles, Accessories
  • Unit: Nos
  • Items: Samsung A15, Vivo Y17, JBL Earphones

Manufacturing Business

  • Raw Materials: Steel Sheet (Kg), Copper Wire (Meter)
  • Finished Goods: Mixer Grinder (Nos)

Service Business

Usually no inventory, but if selling parts:

  • Items: RAM, SSD, HDD (Nos)

8. Inventory Alteration & Deletion

Alter: Select item → Press Enter → Modify → Save

Delete: Only if no transactions → Alt + D → Confirm

9. Common Mistakes & Best Practices

  • No Stock Group → Messy inventory
  • Wrong unit → Valuation errors
  • Duplicate names → Confusion
  • Wrong opening stock → Incorrect reports

Best Practice: Create Groups → Units → Items (in this order)

10. Revision Notes

  • Inventory = Groups + Units + Items
  • Stock Groups classify items
  • Units measure quantity
  • Items represent actual products
  • No duplicates allowed
  • Order: Groups → Units → Items
