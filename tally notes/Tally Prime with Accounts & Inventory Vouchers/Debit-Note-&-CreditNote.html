1. Introduction

Debit Note & Credit Note are used to record Purchase Returns and Sales Returns after the original Purchase/Sales Voucher has been recorded.

They adjust both accounting books and inventory stock accurately.

2. Debit Note (Purchase Return)

Issued when you return goods to supplier or supplier overcharged.

Accounting Effect: Dr. Supplier | Cr. Purchase Return A/c
Inventory Effect: Stock ↓

Shortcut: Ctrl + F9

3. Credit Note (Sales Return)

Issued when customer returns goods or was overcharged.

Accounting Effect: Dr. Sales Return A/c | Cr. Customer
Inventory Effect: Stock ↑

Shortcut: Ctrl + F8

4. Enable Debit/Credit Notes

Gateway → F11 → Accounting Features → Set to Yes:

  • Use Debit/Credit Notes
  • Use Invoice Mode in Debit/Credit Notes
  • Allow Supplementary Details

7. Debit Note – Practical Steps (Invoice Mode)

  1. Vouchers → Ctrl + F9
  2. Press Ctrl + H → Invoice Mode
  3. Select Supplier
  4. Select Ledger: Purchase Return A/c
  5. Enter returned items with quantity & rate
  6. Save

8. Credit Note – Practical Steps (Invoice Mode)

  1. Vouchers → Ctrl + F8
  2. Press Ctrl + H → Invoice Mode
  3. Select Customer
  4. Select Ledger: Sales Return A/c
  5. Enter returned items
  6. Save

9. Linking with Tracking Numbers

During Debit/Credit Note entry → Press Alt + T → Select Receipt/Delivery Note Tracking No. → Prevents double stock movement
